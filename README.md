# 악기 연습 로그

GitHub Pages에 올려서 사용하는 개인용 악기 연습 관리 PWA입니다.

## 기능
- 진행 예정 / 완료 연습 리스트
- 연습 신규 등록 / 수정 / 삭제
- 완료 처리 시 확인 모달 → 완료 정보 입력 모달 → 최종 등록 확인 모달
- 완료 정보: 완료일, 소요시간, 만족도, 연습 방법, 연습 결과, 다음 연습 계획
- 카테고리 등록 / 수정 / 삭제
- 누적 연습시간, 최근 30일, 최근 7일, 카테고리별 통계
- 브라우저 localStorage 저장

## GitHub Pages 사용
Repository에 파일 전체를 업로드한 뒤 Settings > Pages에서 Source를 Deploy from a branch, Branch를 main / root로 설정하세요.
